# BiliParser
本项目创建初衷是用于爬取动漫网站视频资源，后续会增加除动漫以外的影视搜索。
使用语言：java、android
使用依赖库：
1.Jsoup - html解析、网页请求；
2.Room - 本地数据库；
3.okhttp+Retrofit+RxJava+Dagger - 网络请求；
4.SmartRefreshLayout - 上拉加载、下拉刷新；
5.butterKnife - 快捷声明控件、控件点击事件；
6.logger - 打印更长更完整的日志；
7.gson - 超好用的json解析工具。

更新日志：
